Noturna

What is Noturna?

Noturna is an extension for the Chrome browser that allows users to customize the appearance of any website, transforming all its elements in order to provide a more accessible experience. When activating Noturna, all elements of the site display a black background, while the texts are presented in white.

How to install?

To install Noturna, just access the extension link in the Chrome Web Store and click install. If you prefer, you can install directly from the github repository.

How to use?

Using Noturna is quite simple! First, click on the extensions icon in Chrome, represented by a puzzle piece. Then select "Options". Choose from the list of available keyboard shortcuts the one that best suits your needs. Then click "Save". Ready! Now, whenever you press keys of the selected shortcut, the extension will be enabled/disabled.
